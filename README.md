# Italian Translators

A small Vue/Vite site with a directory of Italian translators grouped by city
and country.

## Stack

- Vue 3
- Vite
- PrimeVue
- Sass

## Local development

```sh
npm install
npm run dev
```

## Production build

```sh
npm run build
```

## Deployment

The site is deployed to GitHub Pages:
`https://cudrx.github.io/italiantranslators/`
